Public repository:

`https://github.com/kohjunhao/alfred-crypto-ai-companion`

Alfred is a cryptocurrency chatbot designed to make cryptocurrency data more accessible and accurate to both novices and experts.

Setup
1. Create a virtual environment

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run Alfred

```bash
python run.py
```
